## コントリビューション方針

- Codex の JSON ストリームをわかりやすい Markdown へ整形する取り組みを継続。
- メタデータや GitHub レスポンスのフィクスチャを追加しテストカバレッジを拡大。
- README / CHANGELOG を更新し、Jenkins オペレーター向けに新機能を明確化。
