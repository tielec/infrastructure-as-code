# ドキュメント更新レポート

## 更新サマリー

以下のテーブル形式で更新したドキュメントのみをリストアップします：

| ファイル | 更新理由 |
|---------|---------|
| `CONTRIBUTION.md` | docs/ディレクトリ構造を新規追加したため、プロジェクト概要のディレクトリ構造に反映 |
| `ansible/README.md` | メインREADME.mdの役割変更（詳細手順がdocs/配下に分割）を説明文に反映 |
| `jenkins/README.md` | メインREADME.mdの役割変更（詳細手順がdocs/配下に分割）を説明文に反映 |
